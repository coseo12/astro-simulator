# ADR: 1PN 일반상대론 보정 — Schwarzschild 세차항

- 일자: 2026-04-17
- 상태: Accepted
- 관련: P5-A #178, 수성 근일점 세차 43″/century

## 배경

Newton N-body 엔진에 일반상대론(GR) 보정을 추가하여 수성 근일점 세차를 재현한다.
시각화/교육 목적 — 천체역학 정밀 연구가 아닌 "관측 가능한 GR 효과 시각화".

## 후보

| 항목        | A: 1PN 태양 중심 근사     | B: 전체 EIH 1PN | C: geodesic integrator |
| ----------- | ------------------------- | --------------- | ---------------------- |
| 구현 복잡도 | 낮음 (20줄)               | 중간 (~100줄)   | 높음 (metric tensor)   |
| 정확도      | ±5% (시험입자 근사)       | ±1% (다체 포함) | ±0.1%                  |
| 성능 영향   | O(N) 추가                 | O(N²) 추가      | O(N) per timestep      |
| 검증 용이   | 2-body 해석해와 직접 비교 | 어려움          | 매우 어려움            |

## 결정

**A (1PN 태양 중심 근사) 채택.**

이유:

1. **목적 부합** — 수성 근일점 43″/century를 ±5%로 재현하면 시각화/교육 목적 충분
2. **검증 용이** — 이론치(δω = 6πGM/(a(1-e²)c²))와 직접 비교 가능
3. **성능 최소** — O(N) 추가 비용. N=10000에서도 무시 수준

보정 공식:

```
a_GR_i = (GM₀/(c²r³)) × [(4GM₀/r - v²)r + 4(r·v)v]
```

여기서 body 0 = 태양(가장 무거운 body), r/v = body i의 태양 기준 상대 위치/속도.

## 실측

- 태양+수성 2-body, 100년(415 궤도), dt=60s, Velocity-Verlet
- 결과: **41.46″/century** (이론 42.98″, 오차 3.5%)
- DoD ±5% (40.8~45.1″) **충족**

## 재검토 조건

- 다체 GR 효과(행성 간 1PN)가 필요한 시나리오 발생 → B(EIH) 검토
- GPU 경로에서 GR 지원 필요 → WGSL f32 정밀도로 GR 보정이 의미있는지 먼저 확인
- 블랙홀 시나리오(P5-D 중력렌즈) → 별도 Schwarzschild metric geodesic (C 계열)
