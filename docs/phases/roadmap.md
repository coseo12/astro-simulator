# 확장 로드맵

프로젝트: 웹 기반 천체물리 시뮬레이터
지향: 교육 + 연구 시각화 + 몰입 탐험 + 물리 샌드박스 (모두)
원칙: 진행하며 수정. 각 Phase는 완결된 제품.

---

## Phase 개요

| Phase | 공간 스케일     | 시간 스케일    | 주요 천체                                         | 물리 모델                      | 인터랙션             |
| ----- | --------------- | -------------- | ------------------------------------------------- | ------------------------------ | -------------------- |
| P1    | 태양계 (10^13m) | 실시간 ~ 년    | 행성 8 + 달 + 태양                                | Kepler 해석해                  | 관찰, 카메라 제어    |
| P2    | 태양계 전체     | ~1000년        | +왜소행성, 주요 위성, 혜성, 탐사선, 소행성대 샘플 | Newton N-body (WASM)           | +시간 조절, 파라미터 |
| P3    | 태양계 확장     | ~100만 년      | +카이퍼대/오르트 구름 수십만 개                   | WebGPU Compute N-body          | +궤도 수정 실험      |
| P4    | +근거리 100광년 | ~10억 년       | +항성(Gaia DR3), 쌍성계, 가상 블랙홀              | +일반상대론(렌징, 근일점 이동) | +샌드박스            |

| P6    | +은하계         | 우주 나이 부분 | +은하, 은하단, 상호작용 은하                      | +프리컴퓨트 암흑물질           | +교육 모드           |
| P7    | 관측가능우주    | 138억 년       | +우주거대구조, CMB                                | 제한적 (스냅샷 재생)           | 우주론 모드          |
| P8    | 전체            | 가상           | +웜홀, 이론 천체                                  | 가상 물리                      | 풀 샌드박스          |

## 확장 축

프로젝트는 단일 방향이 아닌 5개 축으로 확장한다.

### 축 1. 공간 스케일

```
태양계 → 근거리 항성 → 은하 국부그룹 → 은하계 → 국부 초은하단 → 관측가능우주
10^13m → 10^17m → 10^22m → 10^21m → 10^24m → 10^27m
```

### 축 2. 시간 스케일

```
실시간 → 일/년 → 천 년 → 100만 년 → 10억 년 → 138억 년
```

시간 역행 포함 (심플렉틱 적분기의 대칭성 활용).

### 축 3. 천체 종류

| 카테고리                                                | 구현 Phase |
| ------------------------------------------------------- | ---------- |
| 태양계 행성/위성/왜소행성                               | P1-P2      |
| 혜성/소행성/탐사선                                      | P2         |
| 카이퍼대/오르트 구름                                    | P3         |
| 항성 (주계열성, 거성, 백색왜성, 중성자별, 펄사, 변광성) | P4         |
| 쌍성/다중성계                                           | P4         |
| 외계행성                                                | P5         |
| 블랙홀 (항성질량/초대질량)                              | P4         |
| 퀘이사, 감마선 폭발                                     | P5         |
| 성운, 성단                                              | P5         |
| 은하 (나선/타원/불규칙)                                 | P6         |
| 은하단, 상호작용 은하                                   | P6         |
| 우주거대구조 (필라멘트, 보이드)                         | P7         |
| CMB                                                     | P7         |
| 가상/이론 (웜홀, 다이슨 구)                             | P8         |

### 축 4. 물리 정확도

```
Kepler 해석해 → Newton N-body → 일반상대론 → 자기유체역학(MHD) → 복사전달
```

검증 가능한 물리 효과:

- 수성 근일점 이동 43"/세기 (일반상대론)
- 빛의 휨 1.75" (중력렌즈)
- 블랙홀 사건의 지평선
- 중성자별 병합 중력파 시각화

### 축 5. 인터랙션 모드

```
관찰 → 파라미터 조절 → 시나리오 실험 → 샌드박스(천체 생성/충돌)
```

- 교육 모드: 설명 오버레이, "만약에" 시나리오
- 연구 모드: 정밀 데이터, CSV 내보내기
- 샌드박스: 사용자 정의 천체, 궤도 파괴 실험

## 확장 설계 원칙

1. **데이터 신뢰성 계층화**: 관측/모델/이론/예술 4단계를 UI에 명시
2. **스케일 전환의 연속성**: "다른 화면"이 아닌 연속 공간으로 느껴지게
3. **코어 재작성 방지**: P1에서 미래 스케일 고려하여 인터페이스 설계. 단, 과잉 설계 금지.
4. **각 Phase = 완결된 제품**: 다음 Phase 없이도 그 자체로 가치

## 로드맵 유연성

이 로드맵은 **진행하며 수정**한다. 규모상 모든 것을 사전 예측할 수 없다.
Phase 완료 시마다 다음 Phase 스코프를 재검토한다.
